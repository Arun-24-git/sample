Find GCD (Euclidean Algorithm)

    Start with two numbers: Let the numbers be num1 and num2.

    Check if the second number is zero:
        If num2 = 0, then num1 is the GCD. Stop.
        If num2 ≠ 0, go to step 3.

    Divide num1 by num2 and calculate the remainder (num1 % num2).

    Update the numbers:
        Set num1 to the value of num2.
        Set num2 to the remainder from the division in step 3.

    Repeat steps 2 to 4 until num2 becomes zero.

    Final Result: When num2 = 0, the value of num1 is the GCD. Output num1 as the GCD.
